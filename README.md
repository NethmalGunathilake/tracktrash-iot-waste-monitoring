# TrackTrash – IoT Based Smart Waste Monitoring System

## About
An IoT-based smart waste monitoring mobile application developed 
using Android Studio as the Final Year Project for the Higher 
Diploma in Computing and Software Engineering at ICBT Campus 
(Cardiff Metropolitan University, UK).

## Features
- Real-time bin fill level monitoring
- Live data display on mobile app
- Automated waste collection scheduling
- Community-level urban waste management
- Data-driven optimization system

## Tech Stack
- Android Studio
- Java
- IoT Sensors
- XML (UI Layouts)

## How to Run
1. Clone or download this repository
2. Open Android Studio
3. Click "Open an existing project"
4. Select the project folder
5. Wait for Gradle build to finish
6. Run on emulator or Android device

## Note
This repository contains the core source code (src folder) 
of the Android application. Open in Android Studio and 
connect to your IoT hardware to run the full system.
